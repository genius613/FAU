Online Forum¶

Please post questions about WEC-Sim on the Issues Page. This forum is managed by the WEC-Sim code development team and users. The issues page is used to interact with the WEC-Sim community, ask questions, and report bugs.

Feature Requests¶

Please post requested feature additions to the WEC-Sim code on the Project Page. This project page is managed by the WEC-Sim code development team and users. The project page is used to encourage future development by users and track the status of requested feature additions.
